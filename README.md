# study_dashboard
a smart study dashboard which will help me keep track of my study hours and help manage my tasks and time and make studyng more engaging
used css, html and javascript
FEATURES;
for productivity:
tasks manager
study planner
sticky notes
focus mode
pomodoro timer
daily goal tracker
to make it fun i also added gamification:
XP system
level progression
study streak 
achivements unlocks
study heatmap like githubs contributions
customization:
(you can also change themes to match your liking)
pink theme
cyber theme
lavender theme
academia theme
forest night(dark green to blue gradient)
animated gradient wallpaper 
effects:
animated rain
stars
floating particles 
analytics:
productivity charts
progress tracking
sctivity visualization
entertainment:
ghibli cafe music (youtube)
smart features:
used a weather api so it gies live weather updates aswell
browser notifications which motivate you every 30 minutes
motivational qoutes
